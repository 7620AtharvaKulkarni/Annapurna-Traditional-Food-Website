# 🍲 Annapurna – Traditional Food Website

## 📖 About
Annapurna is a web‑based project dedicated to showcasing the rich heritage of traditional Indian cuisine. The website aims to preserve authentic recipes, cooking techniques, and cultural stories behind regional dishes.

## 🎯 Description
This website serves as a digital cookbook and cultural archive. It provides users with:
- Detailed recipes from various Indian states
- Historical context and origin of dishes
- Visual guides and flowcharts for easy navigation
- A clean, responsive design built using HTML, CSS, and JavaScript
